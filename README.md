# 4-player-pong
Ez egy 4 playeres pong ami nagyon pog :)

```pins
hangszoró gpio5 = 10
switch gpio4 = 9
y gpio3 = 8
x gpio2 = 7
led gpio15 = 21

Piros joystick értékei magassak  
```
